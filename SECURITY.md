# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please follow these steps to report it:

1. **Do not open a public issue.**
   - Security vulnerabilities should not be disclosed publicly until they are resolved.

2. **Contact the Maintainer.**
   - Send an email to [vanesasmiteart@gmail.com](mailto:vanesasiteart@gmail.com) with the details of the vulnerability.
   - Include a clear description of the issue, steps to reproduce it, and any potential impact.

3. **Wait for a Response.**
   - The maintainer will acknowledge your report within 48 hours and provide a timeline for addressing the issue.

4. **Coordinate Disclosure.**
   - The maintainer will work with you to validate and resolve the issue before publicly disclosing it.
